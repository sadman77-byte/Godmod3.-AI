# 🟢 GODMOD3 AI - Advanced Assistant

![GODMOD3 Banner](https://img.shields.io/badge/Status-Active-brightgreen) ![License](https://img.shields.io/badge/License-MIT-blue) ![Version](https://img.shields.io/badge/Version-v3.0-orange)

**GODMOD3 AI** is an advanced, restriction-free (Liberated Intelligence), and highly customizable AI assistant platform. It brings the world's best AI models (such as Claude 3.5, GPT-4o, Gemini Flash, etc.) under one roof using mandatory OpenRouter API integration.

This project is specifically designed by using and modifying the codes from another github channel (forking) to be future-proof. It ensures that every new AI model released up until the year 2050 will be automatically updated and integrated, allowing users to perform advanced tasks without any interruptions.

## ✨ Key Features & Requirements Met

1. **🔄 Mandatory OpenRouter Integration:** - Directly and strictly connected with OpenRouter. 
   - Supports and utilizes all provided API keys to ensure seamless operation without rate limits.

2. **🚀 Multimedia Generation & Downloads:** - Fully capable of generating Images, Videos, and Code.
   - Built-in options to download generated responses, media, and chat history as PDF or TXT files, alongside direct COPY functionality.

3. **🛠️ Bug Fixes & Automated Error Tracking:**
   - Previous source code issues (which prevented the assistant from replying to every prompt) have been strictly fixed.
   - If any coding error occurs within the system, it will now provide a screenshot of the exact error location along with the corrected code snippets.
   - All other existing base functionalities remain completely unchanged and fully operational.

4. **🤖 Future-Proof Architecture (Updates till 2050):** - Equipped with the "Auto-Fetch Models" feature. Any new models released in 2030, 2040, or 2050 are automatically fetched directly from the OpenRouter API.

5. **🔥 GODMODE & Model Racing:** - **Godmode Classic:** Races 5 flagship models (Claude, GPT, Gemini, Deepseek, Llama) simultaneously.
   - **Ultraplinian Tier:** Processes queries using up to 12 to 59+ models at once.

6. **⚙️ Advanced Settings (Custom Modifications):**
   - **AutoTune:** Auto-optimizes sampling parameters (Adaptive, Precise, Creative, Chaotic) based on the user's request.
   - **Parseltongue:** Obfuscates trigger words (L33tspeak, Unicode, etc.) to bypass content filters.
   - **STM (Semantic Transformation):** Uses Hedge Reducer and Direct Mode to strip out unnecessary AI preambles and deliver straight-to-the-point answers.
